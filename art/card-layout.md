# Card Layout

## Reference
Inspired by Dungeon Mayhem: icons on the left in a vertical strip, title at bottom center, full-bleed art filling the card.

## Fabled Realm Layout Direction
Icons displayed **horizontally** across one end of the card. Title at the **opposite end**. Art fills the middle.

**Option A — Icons top, title bottom:**
```
┌─────────────────────┐
│  ⚔  ⚔  🛡          │  ← card icons (horizontal)
│                     │
│      [ art ]        │
│                     │
│  Channel 🔵1: +1⚔  │  ← channel line (if applicable)
│  CARD NAME          │  ← title
└─────────────────────┘
```

**Option B — Title top, icons bottom:**
```
┌─────────────────────┐
│  CARD NAME          │  ← title
│                     │
│      [ art ]        │
│                     │
│  Channel 🔵1: +1⚔  │  ← channel line (if applicable)
│  ⚔  ⚔  🛡          │  ← card icons (horizontal)
└─────────────────────┘
```

## Zones

### Title
Card name. Opposite end from icons.

### Icon Strip
Horizontal row of card symbols indicating the card's base effect. 1–3 symbols typical.

### Illustration Area
Full-bleed art filling the majority of the card. Character-specific, action-focused.

### Channel Line
Optional. Only present on cards with a Mana-powered upgrade. Shows cost and bonus effect.
Format: `Channel 🔵[cost]: [effect]`
