# Fabled Realm: Friendly Fire

> History's most iconic monsters. D&D archetypes. One winner.

**Fabled Realm: Friendly Fire** is a 2+ player fantasy card game where legendary creatures are reimagined as classic D&D archetypes and thrown into a battle for survival. Play cards to attack, defend, and outmaneuver your rivals using each legend's unique **Fabled Gifts**. The last player standing wins — because in this party, there's no such thing as friendly fire.

## Repo Contents

- `rules/` — Game mechanics, turn structure, card types, glossary
- `characters/` — Full card specs for each legend
- `art/` — Visual style guide and card layout

- `simulations/` — Game simulations
